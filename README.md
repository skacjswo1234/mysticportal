# MYSTIC PORTAL

신비로운 운명과 미래를 탐험하는 마법의 포털 앱입니다.

## 🚀 시작하기

### 설치

```bash
npm install
```

### 개발 서버 실행

```bash
npm run dev
```

브라우저에서 `http://localhost:3000`으로 접속하세요.

### 빌드

```bash
npm run build
```

### 미리보기

```bash
npm run preview
```

## 📁 프로젝트 구조

```
mysticportal/
├── src/
│   ├── components/          # React 컴포넌트
│   │   ├── HomeScreen.jsx   # 홈 화면
│   │   ├── DestinyModule.jsx # 운명 코드 계산기
│   │   ├── MindModule.jsx   # 직관 심리 테스트
│   │   ├── ColorModule.jsx   # 색채 오라 테스트
│   │   ├── PotionModule.jsx # 미래 연금술
│   │   ├── TarotModule.jsx  # 타로 스프레드
│   │   └── Icons.jsx        # SVG 아이콘 컴포넌트
│   ├── utils/               # 유틸리티 함수
│   │   ├── AudioEngine.js   # 사운드 엔진
│   │   └── constants.js     # 상수 데이터
│   ├── App.jsx              # 메인 앱 컴포넌트
│   ├── App.css              # 스타일시트
│   ├── main.jsx             # 진입점
│   └── index.css            # 기본 스타일
├── index.html               # HTML 템플릿
├── package.json             # 프로젝트 설정
├── vite.config.js           # Vite 설정
└── README.md                # 프로젝트 문서
```

## ✨ 기능

- **운명 코드 계산기**: 생년월일을 기반으로 한 운명 코드 계산
- **직관 심리 테스트**: 무의식을 통한 미래 패턴 분석
- **색채 오라 테스트**: 색상 선택을 통한 에너지 분석
- **미래 연금술**: 재료 선택을 통한 물약 제조
- **타로 스프레드**: 3카드 타로 리딩

## 🛠️ 기술 스택

- React 18
- Vite
- Tailwind CSS
- Web Audio API
